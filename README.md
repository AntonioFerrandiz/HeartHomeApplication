# HeartHomeApplication
Hearthome es una start-up creada por estudiantes de la Universidad Privada de Ciencias Aplicadas provenientes de la facultad de ingeniería. Nos enfocamos en crear o mejorar soluciones basadas en la experiencia del usuario de su uso diario.
Nuestro proyecto consitste en crear una aplicación para celulares que facilita el proceso de alquiler de casas y departamentos. El cual consta de dos partes una para los dueños, el cual consiste en la publicación de imágenes de los lugares que van a ser alquilados con una breve descripción para que los posibles futuros inquilinos puedan ver y decidir si lo quieren o no; y la otra parte es para los inquilinos la cual consiste en la creación de un perfil por el cual aparecerán tanto sus datos personales (nombre, edad, antecedentes, etc.) como datos de uso anterior de la aplicación (Pago de renta puntual, respeto por los vecinos, etc.).
